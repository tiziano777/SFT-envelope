```python
def inject_worker_middleware(output_dir: Path) -> None:
    """Copy envelope/middleware/ tree into output_dir/middleware/ for daemon coordination.

    Copies:
    - envelope/middleware/worker/ → setup_{name}/middleware/worker/
    - envelope/middleware/shared/ → setup_{name}/middleware/shared/
    - envelope/middleware/__init__.py → setup_{name}/middleware/__init__.py

    This makes the daemon available without manual setup.
    """
    middleware_src = Path(__file__).parent.parent / "middleware"
    middleware_dst = output_dir / "middleware"

    if middleware_src.exists():
        # Remove destination if it exists (idempotent)
        if middleware_dst.exists():
            shutil.rmtree(middleware_dst)
        # Copy entire middleware tree
        shutil.copytree(middleware_src, middleware_dst)
    else:
        raise FileNotFoundError(f"Source middleware not found: {middleware_src}")
```

Then in generate_setup() at line 127, BEFORE `(output_dir / "run.sh").chmod(0o755)`, add:

```bash
#!/usr/bin/env bash
# Auto-generated launch script
# Experiment: {{ config.experiment.name }}
# Technique: {{ config.training.technique.value }}
# Framework: {{ config.framework.backend.value }}
# Generated by FineTuning-Envelope
set -euo pipefail

SCRIPT_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")" && pwd)"
cd "$SCRIPT_DIR"

# Install dependencies if not already installed
if [ ! -f ".deps_installed" ]; then
    echo "Installing dependencies..."
    pip install -r requirements.txt
    touch .deps_installed
fi

echo "Starting training: {{ config.experiment.name }}"
echo "Technique: {{ config.training.technique.value }} | Framework: {{ config.framework.backend.value }}"
echo "Model: {{ config.model.name_or_path }}"

# Step 1: Start worker daemon in background (blocking handshake)
echo "Starting worker daemon..."
python3 -c "
import asyncio
import logging
from pathlib import Path
from envelope.middleware.worker.daemon import WorkerDaemon

logging.basicConfig(level=logging.INFO)
setup_dir = Path('.')
master_uri = '{{ master_uri | default(\"http://localhost:8000\") }}'
api_key = '{{ api_key | default(\"\") }}'
daemon = WorkerDaemon(setup_dir, master_uri=master_uri, api_key=api_key)
asyncio.run(daemon.bootstrap())
" &
DAEMON_PID=$!

# Step 2: Wait for .handshake_done (max 30s)
echo "Waiting for handshake to complete..."
HANDSHAKE_TIMEOUT=30
ELAPSED=0
while [ ! -f ".handshake_done" ] && [ $ELAPSED -lt $HANDSHAKE_TIMEOUT ]; do
    sleep 1
    ELAPSED=$((ELAPSED + 1))
done

if [ ! -f ".handshake_done" ]; then
    echo "WARNING: Handshake timeout — continuing in degraded mode"
fi

# Step 3: Run training
echo "Starting training execution..."
{{ launch_command }}

# Step 4: Mark training complete
echo "Training completed"
touch .training_done

# Step 5: Give daemon time to flush pending events (5s)
echo "Flushing daemon events..."
sleep 5

# Step 6: Terminate daemon
kill $DAEMON_PID 2>/dev/null || true
wait $DAEMON_PID 2>/dev/null || true

echo "Setup complete"
```

This template:
- Starts WorkerDaemon in background with blocking handshake
- Waits for .handshake_done file (30s timeout, continues on failure)
- Runs training ({{ launch_command }})
- Writes .training_done marker
- Flushes daemon for 5 seconds (allows AsyncPusher to drain queue)
- Terminates daemon gracefully

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Master ↔ Worker | Worker sends experiment state via HTTP/SSH; Master validates API key |
| Worker Daemon ↔ Training Process | Daemon writes state files in setup_dir; training process reads them |
| Generated Scaffold ↔ Filesystem | Middleware code is trusted (comes from envelope/); no user code execution in daemon |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T7-01 | Tampering | run.sh daemon startup | mitigate | Daemon runs as same user as training; state files use atomic writes (Phase 6) |
| T7-02 | Denial | .handshake_done wait timeout | mitigate | 30s timeout with degraded mode fallback; training continues even if handshake fails |
| T7-03 | Information Disclosure | Worker state files (.exp_id, .worker_state.json) | mitigate | State files stored in setup_dir with same permissions as training output; no secrets in state files |
| T7-04 | Elevation | Injected worker code in scaffold | accept | Middleware code comes from checked-in envelope/; no eval or dynamic imports |
| T7-05 | Repudiation | Daemon event queue | mitigate | transfer_log.jsonl provides append-only audit trail per Phase 6 design |
| T7-06 | Denial | AsyncPusher retry loop | accept | Bounded by max_retries (configurable); won't block training execution |

</threat_model>
