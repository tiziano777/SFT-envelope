## Implementation Details

All endpoint signatures follow Pydantic envelope pattern:

```python
@app.post("/handshake")
async def handshake(req: HandshakeRequest) -> HandshakeResponse:
    """Determine strategy and return response."""
    tracer = get_tracer()

    with tracer.start_as_current_span("master.api.handshake.lookup"):
        existing_exp = await repo.find_by_hashes(...)

    with tracer.start_as_current_span("master.api.handshake.strategy"):
        strategy = determine_strategy(existing_exp, req)

    return HandshakeResponse(strategy=strategy, exp_id=exp_id)
```

All errors return semantic responses:

```python
if not experiment:
    raise HTTPException(status_code=404, detail="Experiment not found")
if config_hash_conflict:
    raise HTTPException(status_code=409, detail="Config conflict detected")
```
